This sticky footer example is implemented using html, css and javascript. When page gets loaded or browser window is resized, Javascript function stickFooterToBottom() runs and calculate the height of view port and body content. If the height of viewport is greater than body height then sets body height to vieport's height and places footer content at the bottom of the body with position absolute ELSE places the footer content just below the main-content div and set position static for footer div.
